Neemo - React - Next Documentation

Introduction

Neemo is a Pure React and Next JS template for Digital Business, SEO, and IT Services company websites. This can be used for digital and marketing agencies who provide IT Services, Digital Marketing, SEO, SMM, SEM, or SaaS Applications based services clients. Also, this template is suitable for Mobile App, Software, startups, WebApp, SaaS, and any other business agencies.

Setup Development Environment

In order to kickstart the development of your web app with Neemo, the very first thing you need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Preparation

You'll need to install Node.js >=v14.17+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. powershell or terminal and navigate to the project(Neemo) root. This is powershell in windows and terminal on mac/linux.

Let's looking how setup:

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm run dev

It'll return something like this in console:

> project@2.1.0 dev drive:\project

> node server.js

[ wait ] starting the development server ...

[ info ] waiting on http://localhost:3000 ...

[ ready ] compiled successfully

[ info ] ready on http://localhost:3000

Now in the browser go localhost:3000

For Production Build

Run : npm run build

Then Run : npm run start

See the official next.js doc here

Fonts Used

By default, the template loads Roboto font from Google Web Font Services, you can change the font with the one that suits you best. Google Fonts

You will find the font code in the "style.scss file: public/css/style.scss


$main-font-family: 'Roboto', sans-serif;
$main-color: #316cbf;
$optional-color: #fbe6d4;
$white-color: #ffffff;
$black-color: #202647;
$paragraph-color: #6a6c72;


Note: After SCSS file run then it will be work
